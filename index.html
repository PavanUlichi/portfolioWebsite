Project Description
Service Integrator is a web based integration platform that enables seamless integration among various applications. It has highly configurable processing engines to tackle various integration challenges. It allows millions of messages and files that are exchanged across multiple systems at a time. High volume jobs are processed using file based interfaces. It supports rich set of in-built adapters like FTP/SFTP, SMTP, SCP, JMS, MQ, TCP-IP and SOAP and easy-to-use rules definition framework, enabling rapid integration with Financial Networks such as SWIFT, FIX, Link Up Markets and Feeds such as Reuters, Bloomberg, Telekurs. Configuration changes are flexible since it is incorporated through XML.

Projects

Shop with a snap

This project is made as a part of Hack OHI/O, a hackathon event that happened at Ohio State University. Shop with a snap is a web application built on cloud technology used to shop online with a single click. We used JavaScript, Ajax, AWS EC2, node.js technologies for creating this application. We used Google cloud vision API provided by Google which helps in detecting label from an image, this is purely based on Artificial Intelligence. We also used Amazon product API for retrieving search results using the keyword.

Implementing a RESTful web-service

This project implements a simple calculator application as a RESTful web service. Rest API is one of the popular web-services in the world. This code implements Rest API for performing basic arithmetic operations like addition, subtraction, multiplication, and division. I have used pythons flask framework to implement APIs. The output JSON data is dynamically displayed on the same webpage using AJAX.

Matching Game

This game is really funny to play. When the game is started, it displays two sets of smileys dynamically on either side. The gamers skill lies in identifying an extra smiley on the left-hand side and clicking on that extra smiley. If a correct smiley is selected, the gamer will be prompted to the next level, else the game ends! This project uses simple Javascript and DOM model to implement its functionality.
